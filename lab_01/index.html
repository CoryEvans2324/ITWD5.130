ITWD5.130: WEBSITE DEVELOPMENT

COURSE INFORMATION

Course Description

This course is to provide students with the knowledge and skills to create a static website that meets a specific client brief

Students learn client-side techologies (HTML, CSS, JavaScript) to develop web pages.

Read more about Bachelor of Computing Systems

Number of Credits: 15
Aim: To provide students with the knowledge and skills to create a static website that meets a specific client brief.
Pre/Co Requisite: None


Course objectives & schedule

TIME LEARNING OUTCOMES LEARNING STRATEGIES
5 weeks LEARNING OUTCOME 1
Create a professional website specification document that meets a client design brief.
  • Client Requirements
  • Website specification documents
  • Work flow practices in website design and development
  • Website design: wireframe, sitemap, mocks-up
  • Lectures
  • Tutorials
  • Practical exercises
  • Group discussions
8 weeks LEARNING OUTCOME 2
Apply website design principles and practices to create effective static HTML pages that meet client specifications.
  • Website design: principles & practices
  • Client-side web technologies: HTML5, CSS3, JavaScript
  • HTML and CSS validation
4 weeks LEARNING OUTCOME 3
Explain and apply professional, legal and ethical principles relevant to website design.
  • Legal and Ethical principles and issues
  • NZ Privacy Act. NZ Copyright and Amendments
  • Creative Commons Licences
  • Web Accessibility
